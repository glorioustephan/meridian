# meridian

`meridian` is the author-facing package for Meridian.

It exports:

- `Component<Props>`
- `Primitive<T>`
- `@state`
- `@ref`
- `@effect`
- `@effect.layout`
- `@use`

This package is designed to be consumed from application code. Meridian source must still be compiled before execution with `@meridian/cli` or `@meridian/compiler`.

Documentation:

- Guide: https://glorioustephan.github.io/meridian/guide/introduction
- API: https://glorioustephan.github.io/meridian/api/component
