# @meridian/compiler

`@meridian/compiler` exposes Meridian’s programmatic compile pipeline.

Use this package when you need to transform Meridian source from a custom build tool, editor integration, or test harness.

Most applications should prefer `@meridian/cli`.

Documentation:

- Guide: https://glorioustephan.github.io/meridian/guide/introduction
- API: https://glorioustephan.github.io/meridian/api/compiler
