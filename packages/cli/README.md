# @meridian/cli

`@meridian/cli` provides the `meridian` command-line tool.

Commands:

- `meridian build`
- `meridian watch`

The CLI compiles Meridian source files into standard React TSX before your bundler runs.

Documentation:

- Guide: https://glorioustephan.github.io/meridian/guide/installation
- API: https://glorioustephan.github.io/meridian/api/cli
